Required PHP version needs to be equal or greater than 5.6.
Before launching the update package we recommend disabling all cron actions.
You will be redirected to Home page when update is completed.

During the update:
- the Backup module will be removed
- all notifications will be removed due to structural changes of this module
- uitype10 labels for fields might change, as well as search mechanisms in : Payments in, Payments out, Letters in, Letters out, Reservations
- The "Email templates" module will be substituted by a new one, so all templates and actions related to it will also be removed
