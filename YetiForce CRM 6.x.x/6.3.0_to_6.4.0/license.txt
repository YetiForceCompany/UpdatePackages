Required PHP version for the update package is PHP 7.4.x, 8.0.x.
The update time might be longer depending on the server speed and amount of data in the system.
Before launching the update package we recommend disabling all cron actions.
Log files should be checked after update.

Notes:
- <strong>Stary moduł haseł (OSSPasswords) zostanie pernamentnie usunięty, jeżeli nie chesz ustracić danych, zmigruj je do nowego modułu haseł (Passwords).</strong>
  Zobacz: https://github.com/YetiForceCompany/YetiForceCRM/discussions/15538#discussioncomment-1793851
- Funcjonalność zależności słownikowych uległa przebudowaniu. Wsyzstkie istniejące wpisy zostały usunięte. System spróbuje przywrócić zależności, jednakże należy całość zweryfikować po aktualizacji.
- Moduł SMSNotifier uległ modernizacji w znacznym stopniu. Relacje M-M pomiędzy tym modułem zostaną usunięte. Wpisy w tym module nie zostana usuniete, aczkolwiek zaleca sie je pernamentnie usnąc po aktualizacji.
- W trakcie aktualizacja wykonywane będą operacja na głównych tabelach bazy danych. Jeżeli posiadasz dużą baszę danych, rozważ przeprowadzenie aktualizacji z poziomu CLI, aby uniknąć przekroczenia różnego rodzaju limitów czasowych.
  Zobacz: link do instrukcji z CLI
- Check changes: https://yetiforce.com/en/knowledge-base/documentation/developer-documentation/category/changes-2
- After the update process finishes, please also update all the languages in your system.
